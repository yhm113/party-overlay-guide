이게 뭔가요

PC 앱과 카톡 봇이 같은 클라우드(서버)를 함께 봅니다. 어디서 파티를 만들든 앱·PC·봇에 모두 나타나고, 참가·출발·전적이 자동으로 공유돼요.

PC 앱미니창
→
☁ 서버파티 · 전적
←
카톡 봇/앱파티
💡 봇 → 앱은 즉시 반영, 앱 → 봇은 카톡방에 자동 공지(약 15초)돼요.
1

시작하기 — 계정 만들기 (최초 1회)

파티를 만들거나 참가하려면 계정이 필요해요. 셋 중 하나면 됩니다.

A
봇에서 바로 가입 (앱 없어도 OK)
/앱파티 가입 캐릭터명 → 복구코드가 나와요. 꼭 저장!
B
앱 계정을 봇에 연결 (앱을 먼저 썼다면)
앱 설정탭의 복구코드로 → /앱파티 연동 XXXX-XXXX
C
복구코드 다시 보기 → /앱파티 코드
🔑 봇 가입으로 받은 복구코드를 나중에 앱 설정탭에 입력하면, 봇에서 쌓은 파티·전적이 앱에 그대로 이어져요. (같은 계정으로 통합)
2

카톡 봇 사용법 — /앱파티

참가·나가기·출발·완료·확인은 먼저 /앱파티로 목록을 봐야 번호가 잡혀요.

명령어설명
/앱파티열린 파티 목록 (번호 확인)
/앱파티 생성 카브 캐릭파티 만들기 (카브=카브락일)
/앱파티 참가 1 캐릭1번 파티에 참가
/앱파티 확인 11번 파티 멤버 상세 보기
/앱파티 출발 1출발! 멤버에게 알림 방장
/앱파티 완료 1전적 기록 + 파티 종료 방장
/앱파티 나가기 1나가기 방장이 나가면 해산
/앱파티 전적내 전적 (캐릭별 주간·누적)
/앱파티 도움말전체 명령어 보기

레이드 입력은 번호(1) · 이름(카브락일) · 별칭(카브,카일) 다 인식해요.

나: /앱파티 생성 카브 사르시안 🎵 앱 파티 생성! [앱 파티] 방제 : 카브락일 인원 : 1/8 - 사르시안 - - ...
3

PC 앱 사용법

화면 위에 항상 떠 있는 미니창이에요. 게임 하면서 파티를 확인할 수 있어요.

📑 4개 탭

파티 · 만들기 · 전적 · 설정

📌 핀 버튼

우상단 — 항상 위 켜기/끄기 (게임 위에 띄우기)

🗂 트레이

작업표시줄엔 안 보이고, 우하단 트레이 아이콘으로 관리

✖ 창 닫기

종료가 아니라 트레이로 숨김. 완전 종료는 트레이 우클릭 → 종료

  • 만들기 탭: 레이드 선택 + 캐릭터 → 파티 생성
  • 파티 탭: 참가 · 출발 · 완료 · 취소 버튼
  • 전적 탭: 캐릭터별 주간 · 누적 기록
  • 설정 탭: 닉네임 · 캐릭터 · 복구코드(봇/다른 기기 연결용)
4

이렇게 같이 써요

1
A가 PC 앱(또는 봇)에서 카브락일 파티 생성 → 카톡방에 🎵 새 파티 자동 공지
2
B가 봇에서 /앱파티 참가 1 → A의 PC 앱 인원이 즉시 2/8로 증가
3
방장이 출발 → 멤버 전원에게 알림 + 카톡방에 참가자 목록 공지
4
클리어 후 완료 → 참가자 전원 전적 자동 기록
!
